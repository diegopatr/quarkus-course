# Programação Web Java

Bem-vindo ao repositório da disciplina **Programação Web Java**. Este repositório centraliza os exemplos práticos, exercícios e materiais de apoio desenvolvidos durante as atividades letivas.

## 🎯 Sobre a Disciplina

A disciplina aborda o desenvolvimento de aplicações web modernas utilizando o ecossistema **Jakarta EE** com o framework **Quarkus**. O conteúdo programático contempla a construção de APIs RESTful, persistência de dados com JPA e o desenvolvimento de interfaces com Jakarta Faces.

### Tecnologias e Conceitos
- **Java 21**: Uso de Records, Streams e Virtual Threads.
- **Quarkus**: Stack nativa para ambientes cloud-native.
- **Jakarta REST** (JAX-RS): Especificação para serviços web.
- **Jakarta CDI**: Injeção de dependências e gerenciamento de contextos.
- **Jakarta Persistence** (JPA/Hibernate): Mapeamento objeto-relacional.
- **Jakarta Faces** (JSF): Framework de componentes para interface.

## 🏗️ Estrutura do Repositório

O repositório está organizado em módulos numerados por aula. Cada diretório (`aula-XX`) constitui um projeto Maven independente, o que permite a importação isolada do módulo de interesse.

- `aula-02/`: Fundamentos do Java moderno e manipulação de fluxos com Streams API.
- `aula-03/`: Desenvolvimento de APIs REST seguindo a arquitetura em camadas (Resource, Service, Repository).
